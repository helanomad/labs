# Hela Nomad Labs (labs.helanomad.com)
Showcase site for tools and experiments by Hela Nomad LABS.

## Local preview
This site relies on the GitHub Pages build (remote_theme). For local dev you can install Jekyll, but it’s optional.

## Deployment
- Enable GitHub Pages (Settings → Pages → Source: `main` branch)
- Set Custom domain: `labs.helanomad.com`
- Ensure a `CNAME` file exists with that value
